Mamesaver

Mamesaver is a Windows screen saver which playes MAME ROMS for a specified time.

Requirements

  • Microsoft .NET 2 Framework
  • A working setup of MAME (preferably MAME UI)
  • Some ROMS

Download

  • Mamesave v1.0 Beta

Beta
Please note that this is still beta software and is therefore bound to have a bug or two. If you are happy to test this on your system, any issues would be appreciated.

Open Source

You can check out the project page at https://github.com/mika76/mamesaver
